# DesktopPet_Terraria
该项目是基于 C++ 和 Qt 框架开发的一款Terraria桌面互动宠物。
## ✨ 核心功能
* **桌宠角色**：
  * **小怪/Boss**：彩虹史莱姆、世花、骷髅王、神吞、至尊女巫，灾厄、突变体
  * **NPC**：护士、军火商、派对女孩、哥布林工匠、强盗、税收官、操作员
* **物理反馈引擎**：支持丝滑拖拽、抛掷互动。
* **多实体交互**：支持多个角色同时存在于桌面并且某些角色之间有关联可交互。
  * **军火商&护士**：靠近会转身面向对方冒爱心
  * **哥布林工匠和强盗**：靠近时强盗的头顶会冒出金币，同时哥布林头顶会冒出生气表情
* **专属特效与机制**：
  *  **神明吞噬者**：话痨虫属性大爆发！（不是）
  *  **世纪之花**：二阶段变形与“↑→→→↑→↓↓↓↓↓↓↓”。
  *  **至尊女巫灾厄**：专属“深渊炙炎”与“硫火飞弹”弹幕。
  *  **突变体**：全屏极速穿透冲刺与残影特效。
